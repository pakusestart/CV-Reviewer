## 🚀 Overview

**CV Reviewer 2.0** is a web-based application designed to help job seekers optimize their resumes using the power of Large Language Models (LLMs). This tool provides instant feedback on CV structure, keyword optimization, and overall impact, ensuring your profile is both ATS-friendly and compelling to recruiters.

### Key Features:
- **Instant Analysis:** Deep-dive feedback on your resume's content and layout.
- **ATS Optimization:** Identifies missing keywords relevant to your target industry.
- **Actionable Advice:** Provides specific suggestions to improve bullet points and professional summaries.
- **Responsive Design:** Seamless experience across desktop and mobile devices.

## 🛠️ Tech Stack

- **Frontend:** React.js & Tailwind CSS
- **AI Engine:** Gemini API via Google AI Studio
- **Framework:** Next.js / Vite (depending on your setup)

## 💻 Getting Started

Follow these steps to get a local copy up and running.

### Prerequisites
* **Node.js** (Latest LTS recommended)
* **Gemini API Key** (Get it from [Google AI Studio](https://aistudio.google.com/))

### Installation & Local Development

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/cv-reviewer-2.0.git](https://github.com/your-username/cv-reviewer-2.0.git)
    cd cv-reviewer-2.0
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Configure Environment Variables:**
    Create a `.env.local` file in the root directory and add your API key:
    ```env
    GEMINI_API_KEY=your_actual_api_key_here
    ```

4.  **Launch the app:**
    ```bash
    npm run dev
    ```
